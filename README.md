## README

1. Install NodeJS.
1. Install yarn.

```
npm i -g yarn
```

1. Install node_modules

```
yarn
```

1. Start dev

```
yarn dev
```

## Build

1. Install NodeJS.
1. Install yarn.

```
npm i -g yarn
```

1. Install node_modules

```
yarn
```

1. Build project

```
yarn build
```

1. Push to github pages

```
npx gh-pages -d dist -b gh-pages
```
